# osmium-tool-docker

Docker image that wraps up a default installation of [`osmium`](https://osmcode.org/osmium-tool/).

## Building

```
docker build -t stationa/osmium-tool:latest .
```

## Running

```
docker run -it stationa/osmium-tool:latest <OSMIUM-COMMAND> ...
```
